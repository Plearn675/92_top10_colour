# Top 10 Most Common Colours — Project 92

A simple Flask web application where users can upload an image and receive the top 10 most common colours extracted from it.

This project is part of Day 92 of Angela Yu’s "100 Days of Python" course.

## Features
- Upload any JPG/PNG image
- Automatically analyse the image
- Display the top 10 most common colours
- Clean project structure ready for deployment

## How to Run
1. Create a virtual environment:
   python -m venv venv
2. Install dependencies:
   pip install -r requirements.txt
3. Run the Flask server:
   python main.py

## Technologies Used
- Python 3
- Flask
- Pillow (PIL)
